# Contributing

## Setup

```bash
bun install
bun run test
```

## Generated assets

The following files in `packages/assets-humation-1` are generated by the
Humation asset export pipeline and committed here as source-of-truth
snapshots:

- `manifest.json`
- `assets/**/*.svg`
- `src/embedded.ts`
- `src/manifest-json.ts`
- `src/manifest.ts`

Prefer changing assets through the export pipeline so the manifest, embedded
SVG module, and raw SVG files stay in sync. `packages/assets-humation-1/src/assets.test.ts`
guards their internal consistency (manifest validity, part counts, and that
every `svgPath` resolves to a real file).

Renderer code in `packages/core` is normal hand-written source and welcomes
direct changes.

`r/*.json` is built from `registry/` — edit the sources under
`registry/blocks/` and run `bun run registry:build` to regenerate.

## Before sending a PR

```bash
bun run typecheck
bun run test
bun run pack:smoke
```
